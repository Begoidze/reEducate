1) რამდენი ცვლადი გვაქვს ჯავასკრიპტში.(პასუხი თეორიულად გაეცი)
JavaScript-ში 3 სახის ცვლადი გვაქვს:
    var   ძველი, function-scoped, აღარ გამოიყენება
    let   block-scoped, შეიძლება შეიცვალოს
    const block-scoped, განსაზღვრის შემდეგ ვეღარ შევცვლით

2) რამდენიტიპი გვაქვს ჯავასკრიპტში.(ჩამოთვალე)(პასუხი თეორიულად გაეცი)

პრიმიტიული (7):
    1. String
    2. Number
    3. BigInt
    4. Boolean
    5. Undefined
    6. Null
    7. Symbol
არაპრიმიტიული (1):
    8. Object (მასივი, ფუნქცია...)

3) რომელი მეთოდი აქცევს სტრინგს მასივად.(პასუხი თეორიულად გაეცი)

    სტრინგს მასივად აქცევს split() მეთოდი, რომელსაც გადავცემთ რომელი სიმბოლოს
    მიხედვით გაყოს სტრინგი
    მაგალითად:
    let skills = "JavaScript backend";
    let parts = skills.split(" "); // ["JavaScript", "backend"]

    თუ არაფერს გადავცემთ:
    skills.split() // ["JavaScript backend"] - მთელი სტრინგი ერთ ელემენტად

    თუ ცარიელ სტრინგს გადავცემთ:
    skills.split("") // ["J","a","v","a","S","c","r","i","p","t"," ","b","a","c","k","e","n","d"] - ყოველი სიმბოლო ცალ-ცალკე


4) სტრინგი პრიმიტიული ტიპია თუ არა ? .(პასუხი თეორიულად გაეცი)
სტრინგი პრიმიტიული ტიპია, JavaScript-ში სტრინგი ინახება მნიშვნელობით (by value) და
არა მითითებით (by reference).

5) ჩამოთვალე რა მეთოდები ვისწავლეთ მაგ -> length(პასუხი თეორიულად გაეცი)
- length        -> აბრუნებს სტრინგის სიგრძეს
- slice()       -> ამოჭრის სტრინგის ნაწილს (start, end)
- charAt()      -> აბრუნებს სიმბოლოს მითითებული ინდექსით
- toUpperCase() -> ყველა ასოს დიდად აქცევს
- toLowerCase() -> ყველა ასოს პატარად აქცევს
- concat()      -> აერთებს ორ ან მეტ სტრინგს
- split()       -> სტრინგს ყოფს მასივად
- replace()     -> პირველ დამთხვევას ცვლის სხვა ტექსტით
- replaceAll()  -> ყველა დამთხვევას ცვლის სხვა ტექსტით
- trim()        -> თეთრ სივრცეებს შლის დასაწყისიდან/ბოლოდან
- includes()    -> ამოწმებს შეიცავს თუ არა სტრინგს (true/false)
- indexOf()     -> პოულობს სიმბოლოს/სიტყვის პირველ ინდექსს

loop-ები:
- for loop   -> იმეორებს კოდს განსაზღვრული რაოდენობით
- while loop -> იმეორებს კოდს სანამ პირობა true-ია

ternary operator:
- პირობა ? მნიშვნელობა თუ true : მნიშვნელობა თუ false
- isOnline ? "Y" : "N"
